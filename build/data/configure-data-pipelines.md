## What You'll Learn

- How to set up windowed roll-ups and aggregations
- How to create derived metrics from raw captured data
- How to configure cost-efficient querying patterns
- How to manage data retention and lifecycle policies

## What Problem This Solves

Raw captured data grows fast. You need aggregated views (hourly averages, daily counts, trend summaries) that are cheaper to store and faster to query than scanning every raw reading. Data pipelines transform raw captures into actionable metrics.

## Components Needed

- Captured data in Viam's cloud (from Capture and Sync Data)
- Familiarity with SQL/MQL queries (from Query Data)
