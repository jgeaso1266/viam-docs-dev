# Viam Essentials Ep 8: Demo Dashboard

A minimal white-labeled web dashboard for the video shoot.
Built with TypeScript SDK + React (or plain HTML/JS for simplicity).

## Requirements

- Functional login page with custom branding (not Viam logo)
- Fleet overview: list machines, show online/offline status
- Machine detail: live camera feed, recent detections
- Connects to real machines via Viam TypeScript SDK

## Pre-production tasks

1. Design a fake company identity (logo, color scheme) for the
   white-label demo. Something like "AcmeBots" or "Wrench Watch".
2. Build the login page using Viam's OAuth flow with custom branding.
3. Build the fleet overview page.
4. Build the machine detail page with camera stream.
5. Test against the machines used in Episodes 1-7.

## Scope

This is a demo for a 90-second video. It needs to look real and work
against live data, but it does not need to be production-quality code.
Clean UI, functional on camera, connects to real machines.
